Mervin & Mervis

A local chat arena for small, fine-tuned LLMs that run on your own CPU -- no GPU, no API keys, nothing leaves the machine.

Every model answers as two characters at once:

  • Mervin -- a sardonic pessimist.
  • Mervis -- a relentless optimist.

The split personality is baked in by fine-tuning alone -- there is no system prompt at all.

Why it's a challenge

It looks silly, but it is a genuinely hard ask for a small model: from one request, produce two different answers at once -- opposite personalities -- and wrap each in its own custom <Mervin> / <Mervis> tags, every time, with no system prompt to lean on. So the arena is really a test of which models can absorb heavy customization and still think clearly -- rather than dropping a tag, blending the two voices into one, or swapping who says what.

How to use it

Each column to the right is one model. Only one model lives in memory at a time, so the others look dimmed. Click a dimmed column to load that model (a short wait while it warms up), then chat with it in that column. Use the ◀ ▶ arrows up top to slide one column at a time.

The models

Phi-4-mini, Gemma 4 E2B/E4B, Qwen 2.5 7B, and Mistral 7B -- all Q4_K_M GGUF, all Apache-2.0 / MIT licensed, all small enough to fit in 16 GB. They download smallest-first, so the lightest model is ready to chat while the rest are still arriving.
