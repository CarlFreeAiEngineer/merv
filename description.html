Mervin & Mervis

A local chat arena for small, fine-tuned language models that run completely on the web server device.

Every model answers as two personalities at once:

  • Mervin -- a sardonic pessimist, similar to Marvin the Paranoid Android in Hitchhiker's Guide to the Galaxy.
  • Mervis -- a relentless optimist, maybe a little too cheerful.

The split personality is baked in by fine-tuning alone -- there is NO SYSTEM PROMPT AT ALL.

Why this customization?

It looks silly, but it is a genuinely hard ask for a small model: from one request, produce two different answers at once -- opposite personalities -- and wrap each in its own custom <Mervin> / <Mervis> tags, every time, with no system prompt to lean on. So the arena is really a test of which models can absorb heavy customization and still think clearly. If this is too difficult for a model, it will screw up the tags, or the personalities will blend; especially after multiple exchanges.

How to use it

Each column to the right is one model. Only one model lives in memory at a time, so the others look dimmed. Click a dimmed column to load that model, then chat with it in that column. Only one model can fit in memory at a time, and the models files are gigabytes, so the switchover might take several minutes.

Remember that you can't judge the speed of the model until the 2nd response in a row, because the first response is always slower due to loading the model into memory.

Use the ◀ ▶ arrows up top to slide one column at a time.

The models

Phi-4-mini, Gemma 4 E2B, Qwen 2.5 7B, Mistral 7B, and Gemma 4 E4B, all Apache-2.0 or MIT licensed, all small enough to fit in 16 GB. They download smallest-first, so the lightest model is ready to chat while the rest are still arriving.
